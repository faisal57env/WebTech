Character Entity / HTML entites

Used to display the special characters in webpage which are not available in keyboard

We can access the reserved characters like < and > by using character entity.

start with & symbol followed by the name of character and end with ;

~!@#$%^%&^*()

α β © ™ ® ∵ < > ⋒

< h1 > Hello world < / h1 >
