# Web Technologies Learning Repository 🚀

Welcome to my personal learning repository for **Web Technologies**. This repository contains a structured, hands-on collection of source code, practical tasks, and course notes documentations tracking my journey through mastering foundational core frontend technologies, specifically **HTML5** and **CSS3**.

The repository is organized directly into granular topics—ranging from basic semantic layouts to advanced CSS capabilities like Flexbox, Grid, Animations, and Responsive Web Design.

---

## 📌 Repository Structure

The project is cleanly modularized into two distinct core directories:

```text
├── CSS/
│   ├── animations/          # Keyframe animations, rotations, and zooming effects
│   ├── backgroundproperties/# Background colors, images, and Parallax implementations
│   ├── box model/           # Padding, borders, margins, and content sizing
│   ├── color/               # Hex, RGB, RGBA, HSL, and HSLA coloring architectures
│   ├── cursor/              # Native UI mouse cursor tracking states
│   ├── display/             # Block, Inline, Inline-Block, Visibility states
│   │   └── flexbox/         # Detailed Flexbox layouts (grow, shrink, wrap, alignment)
│   ├── filter/              # Graphical image filtering effects (blur, contrast, grayscale, etc.)
│   ├── fontproperties/      # Typography, font weights, styles, and custom web fonts
│   ├── gradients/           # Linear, Radial, and Conic gradient architectures
│   ├── grid/                # Modern CSS Grid layouts (rows, columns, explicit templates)
│   ├── listproperties/      # Customized navigation structures and markers
│   ├── mediaQueries/        # Responsive breakdown layers for cross-device styling
│   ├── opacity/             # Native alpha opacity layers and translucent backdrops
│   ├── overflow/            # Content bounds handling (scroll, hidden, auto)
│   ├── positions/           # Layout positioning layouts (static, relative, absolute, fixed, sticky)
│   ├── selectors/           # Universal, group, attribute, combinators, and pseudoclass structures
│   ├── tasks/               # UI Component assignments (Animated buttons, flying labels)
│   ├── textproperties/      # Spacing, formatting alignments, decoration layers
│   ├── transform/           # 2D and 3D spatial manipulations (rotate, scale, translate, skew)
│   ├── transition/          # Smooth state timing interpolations
│   ├── types/               # Inline vs Internal styling demonstrations
│   ├── zindex/              # Explicit 3D stacking contexts
│   └── notes.md             # Consolidated technical roadmap tracking
│
└── HTML/
    ├── Day1/                # Basic heading configurations, text formatting, and resumes
    ├── Forms/               # Interactive input schemas, custom inputs, datalists, and actions
    ├── Hyperlinks/          # Internal, external, media anchors, and assets processing
    ├── Lists/               # Ordered, unordered, and descriptive nested trees
    ├── character entity/    # Reserved HTML parsing entities and typography shortcuts
    ├── mediatags/           # Native rich media layers (Audio, Video, Pictures, Poster attributes)
    ├── pagination/          # Client-side multi-page linear cross-linking maps
    ├── semantic/            # SEO-friendly vs classic layout elements
    └── tables/              # Structural grids, spans (colspan, rowspan), and data arrays
