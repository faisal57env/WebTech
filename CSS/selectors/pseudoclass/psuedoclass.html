Pseudo class selectors

it is used to target the special state of an element

All the Pseudo class selectors are represented with :

Dynamic pseudo class selectors

  • link pseudo class selector
  • visited pseudo class selector
  • active pseudo class selector
  • focus pseudo class selector
  • hover pseudo class selector
  • checked pseudo class selector

Structural pseudo class selectors

  • first-child pseudo class selector
  • last-child pseudo class selector
  • nth-child pseudo class selector
  • first-of-type pseudo class selector
  • last-of-type pseudo class selector
  • nth-of-type pseudo class selector
